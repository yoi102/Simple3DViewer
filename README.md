**Simple3DViewer — 基于 ODA SDK 的简易 3D 查看器Demo**



**遗憾点**

- 暂不支持在文件读取过程中中断/取消；一旦开始加载需等待完成。

**主要特性**

- 支持常见格式：DWG / DGN / STEP / STP / STL / OBJ / VSF / VSFX / PRC
- 基本交互：旋转、平移、缩放
- 可使用../TestFiles/目录下文件测试
